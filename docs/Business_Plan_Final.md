## 执行摘要

### 一句话项目定位

**智汇大棚是一套面向高附加值作物（藏红花、人参、铁皮石斛、名贵兰花等）设施农业的"数字孪生+智能调控"一体化解决方案，通过"感知-诊断-决策-执行"闭环，实现从"靠经验种植"到"靠数据种植"的跨越。**

### 三大核心痛点

| 痛点 | 传统现状 | 损失量化 |
|------|----------|----------|
| **监控盲区** | 人工巡检3次/天，夜间/恶劣天气无人值守 | 一次供暖故障损失60万元 |
| **调控粗放** | 凭经验调控，温湿度波动大 | 品质降级导致出口订单损失30%+ |
| **多棚失控** | 20座以上温室数据分散，异常难定位 | 人力成本年增40%，事故率反升 |

### 解决方案闭环

```
传感器数据采集 → AI数字孪生建模 → 三级告警识别 → SOP任务派发 → 远程精准控制 → 全链路留痕复盘
```

### 核心数据成果

- **技术参数**：WebSocket 5秒级实时推送、Three.js 3D数字孪生、FastAPI毫秒级响应
- **实测效果**：24小时连续监控、异常响应从2小时压缩至5分钟
- **成本优势**：相比进口Plant Factory系统（50-200万），成本降低90%+

## 二、产品与技术方案

### 2.1 产品形态

**一句话描述**："一个驾驶舱 + 一套数字孪生 + 四类核心功能"

#### 2.1.1 驾驶舱（Dashboard）—— 30秒掌握全局

**已实现功能**（基于 `/dashboard` 页面）：

| 功能模块 | 技术实现 | 用户价值 |
|----------|----------|----------|
| **5大指标卡** | 顶部HUD设计 | 大棚总数、活跃告警、曲线吻合度、当前作物、系统状态一目了然 |
| **24小时环境曲线** | ECharts + WebSocket实时推送 | 温度/湿度/土壤水分三曲线同步，支持历史回溯 |
| **Plant Factory 3D** | Three.js 1:1虚拟大棚 | 可视化展示作物生长状态，支持鼠标交互 |
| **棚群监控列表** | 风险评分算法 | 多温室风险排序，优先处理高危大棚 |
| **作物生长进度** | 生长模型计算 | 生长天数、预计采收倒计时 |
| **实时告警面板** | 三级告警（紧急/预警/提示） | 红色闪烁紧急告警，一键确认处理 |
| **SOP任务列表** | 定时任务+条件触发 | 灌溉、通风、遮阳任务自动推送与完成闭环 |
| **远程设备控制** | 权限验证+API控制 | 灌溉60s/180s一键控制，操作留痕 |

#### 2.1.2 棚群管理（Greenhouses）—— 单棚精细化管理

**已实现功能**（基于 `/greenhouses` 页面）：

- **多棚列表**：风险评分、曲线吻合度、最新读数概览
- **单棚详情**：阶段管理、告警记录、SOP任务、24小时历史曲线
- **任务执行**：灌溉、巡检、维护任务一键完成
- **批量操作**：批量灌溉、数据导出（开发中）

#### 2.1.3 设置中心（Settings）—— 配置与数据分析

### 2.2 技术架构

#### 2.2.1 技术栈（已实际采用）

| 层级 | 技术选型 | 选择理由 |
